# Tic-Tac-Toe Game

A classic Tic-Tac-Toe game built with HTML, CSS, and JavaScript. This project features a custom welcome screen for player name entry and a responsive, styled game board.

## Features
* **Player Name Entry:** A dedicated landing page to personalize the game.
* **Win Detection:** Automatically identifies and announces the winner based on classic patterns.
* **Reset Functionality:** Easy game restart options via Reset or New Game buttons.
* **Modern UI:** Clean design with a custom color palette and responsive layout.

## Technologies Used
* **HTML5:** Structure of the game and forms.
* **CSS3:** Styling, grid layouts, and responsiveness.
* **JavaScript:** Game logic, win validation, and event handling.

## How to Run
1. Clone this repository.
2. Open `frontpage.html` in your browser to start the game.
3. Enter player names and enjoy!
